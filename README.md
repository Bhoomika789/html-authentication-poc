# Authentication System Styling

This project is a simple Authentication System built using **HTML, Bootstrap 5, and Custom CSS** as part of the CampusPe Fullstack Java Development course.

The application demonstrates a basic authentication workflow with multiple pages including login, registration, password recovery, and a dashboard.

---

## Technologies Used

* HTML5
* Bootstrap 5
* Custom CSS
* Google Fonts

---

## Pages Included

1. **login.html** – User login page
2. **register.html** – New user registration page
3. **forgot-password.html** – Password recovery page
4. **reset-password.html** – Password update page
5. **dashboard.html** – Welcome dashboard after login

---

## Features

* Responsive design using Bootstrap
* Professional card-based UI layout
* Custom CSS styling
* Gradient background
* Button hover effects and smooth transitions
* Navigation between authentication pages
* Dashboard with navbar and logout button
