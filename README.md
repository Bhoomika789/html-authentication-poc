# HTML Authentication System POC

This is a simple Proof of Concept (POC) HTML authentication system created as part of the CampusPe assignment.

## Pages Included

1. login.html
2. register.html
3. forgot-password.html
4. reset-password.html
5. dashboard.html

## Features

- Simple HTML structure
- Redirections using anchor tags
- No CSS or JavaScript used
- Basic form inputs

## How to Run

Open login.html in a browser and navigate through the links.
